Some clarifications on the directions that I could have used

while working on assignment #1

To get to the page, this is the process:

  • In order to see anything, you must be VPN'd to OSU
  • The site needed to answer the actual assignment externally is:
  • http://access.engr.oregonstate.edu:XXXXX
  • XXXXX is the same port that was used to launch the diagnostic.js file
  • For me, I chose http://access.engr.oregonstate.edu:13387
  • Of course, which assumes that port is otherwise unused
  • Alternatively http://flip3.engr.oregonstate.edu:13387 can be used

To verify that the setup works:

  • Launch the VPN connection
  • Launch the remote shell connection
  • Attempt to access this page: http://access.engr.oregonstate.edu:13387 - NOTHING HAPPENS!!!
  • Launch the diagnostic using the following command:

    • ./node_modules/forever/bin/forever start diagnostic.js 13387

  • Relaunch this page: http://access.engr.oregonstate.edu:13387 - "My SQL is working"
  • Stop the forever process using the following command:

    • ./node_modules/forever/bin/forever stopall

  • Relaunch this page: http://access.engr.oregonstate.edu:13387 - again, NOTHING HAPPENS!!!
  • Then restart the forever diagnostic using the following command:

    • ./node_modules/forever/bin/forever start diagnostic.js 13387

  • Relaunch this page: http://access.engr.oregonstate.edu:13387 - again, My SQL is working!!!
